Don't forget! When working with RxJS 6 (or higher), you need to adjust the code in case you're NOT using the rxjs-compat library.

Use

import { Subject } from 'rxjs'; 

instead of 

import { Subject } from 'rxjs/Subject'; 

and 

import { Subscription } from 'rxjs'; 

instead of 

import { Subscription } from 'rxjs/Subscription';
